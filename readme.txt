Hello, follow the guideliness for deployment of the project.

1) Download the following softwares.
	a) NodeJs
	b) MongoDb - as a server. (The default connection is on port 27017, Ip- localhost, no authentication) 

2) open the SocialMedia directory in command prompt
	>> npm install   
	After the successful installation give the next command

	>> npm start
	THe frontend server will get started on the port number 4050	
	
	
3) open the backend folder in SocialMedia directory in command prompt
	>> npm start


4) In the opened login UI start by creating a user.
